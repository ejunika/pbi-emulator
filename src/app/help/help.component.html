What is this?

This is an emulator to test the embedded report of Power BI. Here you can test the embed report with or without applying the RLS (Row Level Security).

Prerequisite

For security reasons, we are not handling user Authentication, and so you need to have a client, which can log in to Power BI and get you back the Azure Access Token. Postman is the recommended client for Authentication steps.

How to use the emulator?

For getting started with the emulator follow the steps mentioned below.

  1. Log in to the Power BI using Postman and get the access token and you are ready to use this application.
  2. Use the access token to log in to the application.
  3. After a successful login, you will be redirected to the home page of the application.

And now you are ready to go.
