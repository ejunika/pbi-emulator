What is this?

This is an emulator to test the embeded report of Power BI. Here you can test the embed report with or without apply the RLS (Row Level Security).

Prerequisite

For security resions we are not handling the user Authentication & Authorization process. There for you need to have a client, which can login to Power BI and get you back the Azure Access Token. I'll recoment Postman for Authentication steps.

How to use the emulator?

For getting started with the emulator follow the steps mentioned below.

  1. Login to the Power BI using Postman and get the access token
  2. Click on Manage Token menu, provide the access token and save it
  3. Click on Manage Account menu, provide the username and save it

And now you are ready to go.
